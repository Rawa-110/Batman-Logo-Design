# Batman-Logo-Design
Description:
This project focuses on creating a 2D Batman logo using Onshape. The sketch was created using basic sketch tools such as lines, arcs, circles, and dimensions. The design was constrained and fully dimensioned to ensure accuracy and symmetry. After completing the sketch, it can be extruded to create a 3D model.

ntroduction

Computer-Aided Design (CAD) plays a vital role in modern engineering and product development by enabling accurate and efficient digital modeling. This project was completed using Onshape, a cloud-based CAD platform, to design a 2D Batman logo and transform it into a 3D model. Throughout the project, essential sketching tools, geometric constraints, dimensions, and extrusion features were applied to create a precise and symmetrical design. The purpose of this exercise was to develop practical CAD skills while improving accuracy, creativity, and familiarity with professional design software.
الهدف (Objective)

To practice CAD modeling skills by designing the Batman logo while applying sketching techniques, dimensions, constraints, and extrusion in Onshape.
